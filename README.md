# kira2diffeq

Small utility to build differential equations for master integrals from Kira reduction output.

It parses a Kira `.m` reduction file plus `masters.final`, computes
∂/∂`x` derivatives of master integrals in Mathematica expressions, and exports
the resulting matrix either as plain text or Mathematica format.

## Repository layout

- `include/kira2diffeq/` — public headers.
- `src/` — library and CLI implementation.
- `tests/` — minimal unit and smoke tests.
- `benchmarks/` — sample benchmark DE outputs.
- `test_fctopology100C/` — captured Kira example inputs used for local checks.

## Requirements

- C++17-compatible compiler
- CMake ≥ 3.16
- `pkg-config`
- `ginac`
- `yaml-cpp`

On Fedora-like systems:

```bash
sudo dnf install pkgconf-pkg-config ginac-devel yaml-cpp-devel
```

## Build

```bash
cd /path/to/kira2diffeq
cmake -S . -B build
cmake --build build -j
```

Run tests:

```bash
ctest --test-dir build --output-on-failure
```

## CLI usage

```bash
./build/kira2diffeq-cli \
  -c <kira_config_dir> \
  -m <kira_math_output_file> \
  -f <masters.final> \
  -t <topology_name> \
  -v <kinematic_variable> \
  -o text|math|all \
  -O <output_file>
```

Required arguments are `-c`, `-m`, and `-f`.  
Optional flags:

- `-v`: derivative variable (default: all kinematic invariants in the topology config)
- `-t`: topology name (default: first topology in the config)
- `-o`: output format (`text`, `math`, `all`; default `text`)
- `-e`: use epsilon form (`d = 4 - 2*eps`) in output
- `-O`: write output to file
- `-h`: show help
